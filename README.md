# Python-Learning-Planner
A Python program that creates a personalized learning plan based on user time and goals.

#How to run it
1. Make sure you have Python installed
2. Download main.py
3. Run: python main.py
4. follow prompts

#Functions
- Collects user information (name, study time, learning goals)
- Validates user input
- Calculates total learning hours
- Generates a customized 4-week curriculum
- Saves the learning plan to a text file
